## Features
- Submit **enquiry forms** with name, email, phone number, and message.  
- Responsive design for both desktop and mobile devices.  
- Easy-to-navigate interface with clear instructions.  
- Form submissions stored for admin review.  

---

## Technologies
- **Frontend:** HTML5, CSS3, JavaScript  
- **Framework (optional):** React.js  
- **Backend (optional):** Node.js / Express  
- **Database (optional):** Google Sheets / Firebase / MongoDB
